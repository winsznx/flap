# flap

> Flap your way to sats-back glory. A flappy-shaped arcade game with cUSD micro-stakes on Celo.

Pay a tiny cUSD stake per play, beat the threshold, get more back. Daily bounty pool sponsored by the community.

## Stack

- Next.js 16 (App Router)
- React 19
- Tailwind CSS v4
- TypeScript

## Develop

```bash
pnpm install
pnpm dev
```

Open http://localhost:3000.
